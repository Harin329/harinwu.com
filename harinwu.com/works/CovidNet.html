CovidNet

Mar 2020



PRODUCT:

Coronavirus Crowdsource Tracker

TECHNOLOGY:

React Native, Python, MongoDB

COMPANY:

Personal Project

DESCRIPTION:

Identifying a lack of testing during a pandemic. I was able to come up with the idea of crowdsource testing weeks before universities, governments and companies. Asympotomatic carriers need to be alerted, but have their privacy kept at the same time. Using a 30 day group backtracking algorithm, the program is able to mark and calculate risk levels for all users. The data is only ever presented to the respective user, to maintain data privacy. The only requirement was a daily 30 second form, indicating any symptoms and places visited. The project was incomplete due to lack of interest in user testing. NOTE: Lack of usage became a problem for both government promoted and open source projects of similar interest.
