BackendRoom

Oct 2021



GOAL:

Creating a template to quickly spin up a full backend.

TECHNOLOGY:

Python, MySQL, Nginx, Shell, Docker

COMPANY:

Personal Project

DESCRIPTION:

Loving the FastAPI, MySQL backend stack. I created this docker-compose template to allow myself to quickly get a backend up and running. It has everything from boilerplate stub REST API calls to the option for let's encrypt SSL certificate setup.
