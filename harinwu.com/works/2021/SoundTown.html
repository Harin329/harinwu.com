See SoundTown in Action!

PRODUCT:

Music sync room for Spotify

ROLE:

Full Stack Developer

TECHNOLOGY:

React, TypeScript, Ant Design, MongoDB, GraphQL, Azure, Github Actions, Figma

COMPANY:

Personal Project

DESCRIPTION:

A 72 hour typescript project focused on exploring Ant Design components. The webapp provides an environment for users to enter, share their currently playing songs and allow others to join in. There is also the option to queue requests and leave little messages for everyone else in the room. The emphasis of the project was to get deeper exposure with some new tools like GraphQL, while focusing on clean design. The Spotify API is heavily used to control many of the playback features, with custom logic used to manage matching up user playback.
