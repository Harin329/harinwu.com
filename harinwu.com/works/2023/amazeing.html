I wanted to learn Rust for a long time, and finally took the initiative to learn it. Following the book Hands On Rust, I created a dungeon crawler game using Rust and WebAssembly. The game is a simple dungeon crawler game where the player has to find the exit of the dungeon. The player can move around the dungeon and fight monsters.

Learning through application is the best way to learn, and this project allowed me to gain a strong understanding of how Rust works, as well its benefits. I am hoping to now further improve my Rust skills by improving the game, and to also apply it in practical domains.
